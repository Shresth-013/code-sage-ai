# 🚀 code-sage-ai

A full-stack AI-powered web application that helps developers with:

- Resume analysis (ATS scoring)
- Code review
- LeetCode hints
- Learning roadmaps

## 🛠 Tech Stack
- Frontend: React + Vite + TailwindCSS
- Backend: Node.js + Express
- AI: Anthropic Claude API
